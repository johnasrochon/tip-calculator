# Tip Calculator Website

## Project Description
This project is a responsive Tip Calculator website that allows users
to calculate the total tip and tip per person based on bill amount,
tip percentage, and number of people.

## Features
- Calculates total tip
- Calculates tip per person
- Responsive design
- Simple user interface

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Instructions to Run
1. Download or clone this repository.
2. Open index.html in any web browser.
3. Enter the bill, tip percentage, and number of people.
4. Click "Calculate" to see results.
